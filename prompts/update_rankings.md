# Задача: обновить рейтинг и очки (приоритет 3)

Ты обновляешь данные теннисного приложения. Работаешь ТОЛЬКО с файлом
`data/rankings.json`. Профили игроков (`data/players.json`) НЕ трогаешь —
там нет рейтинга и очков, они вынесены сюда специально.

## Источник
ATP (atptour.com/rankings) — официальный рейтинг (пересчитывается по понедельникам)
и набранные очки. Tennis Abstract — для сверки.

## Что сделать
1. Прочитай `data/rankings.json` (массив записей по playerId).
2. Для каждого игрока обнови:
   - `current` — текущая позиция в рейтинге ATP;
   - `seasonDelta` — изменение позиции за сезон (целое, может быть отрицательным);
   - `trend` — одно из "up" / "down" / "stable" (динамика за последние недели);
   - `ytdPoints` — очки за текущий год (Year-To-Date);
   - `seasonPoints` — очки сезона по твоей текущей методике (как в существующих данных).
3. Не добавляй и не удаляй игроков — состав игроков меняется в `players.json`
   отдельной задачей. Здесь только цифры по уже существующим playerId.

## Правила
- `trend` строго из трёх значений: up / down / stable.
- Не выдумывай числа — бери с ATP. Не уверен — оставь прежнее значение.
- JSON валидный, UTF-8, отступ 2 пробела.

## После правок
`python3 build_config.py --check` — и если ок, остановись.
